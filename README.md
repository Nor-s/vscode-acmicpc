# vscode-acmicpc

-   vscode-leetcode 의 코드를 백준에 맞게 수정하였다.

## Features

-   문제집 추가 기능
-   단계별 문제 풀이
-   class 문제 풀이
-   id 입력시 푼 문제 체크

## Extension Settings

-   `acmicpc.workbookList` : 문제집 추가
-   `acmicpc.user` : 백준 아이디
